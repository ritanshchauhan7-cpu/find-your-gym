# FIND YOUR GYM

A React-based web app that helps users find gyms based on budget, location, experience level, and fitness goals.
I built this after noticing that many people (including myself) struggle to choose the right gym based on their needs.

##  Features

- Filter gyms by budget, location, and goal  
- Beginner-friendly recommendations  
- Simple UI with smooth animations  
- WhatsApp contact option  

##  Tech Stack

- React.js  
- JavaScript  
- CSS
